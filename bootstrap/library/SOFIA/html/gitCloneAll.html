gitCloneAll {SOFIA}R Documentation

Clone All

Description

Clone all SOFIA-TAF repositories from GitHub.

Usage

gitCloneAll(topdir = "c:/git/sofia-taf", method = "https", tree = TRUE)

Arguments

topdir

local empty directory to clone to.

method

cloning method, either "https" or "ssh".

tree

whether to organize repositories in a nested SOFIA tree, see details below.

Details

Generally, the default method = "https" works best in Windows, while method = "ssh" works best in Linux.

When tree = TRUE, this function will recognize standard SOFIA repository names such as 2022Area37Demo and clone to a nested subdirectory 2022/Area37/Demo instead of 2022Area37Demo.

Value

String vector of cloned repositories.

Author(s)

Arni Magnusson.

See Also

gitRepos fetches a complete list of GitHub repositories.

gitClone clones a repository from GitHub.

SOFIA-package gives an overview of the package.

Examples

## Not run: 
gitCloneAll()
gitCloneAll("~/git/sofia-taf", "ssh")

## End(Not run)
