gitRepos {SOFIA}R Documentation

List Repositories

Description

List all GitHub repositories owned by an organization or user.

Usage

gitRepos(owner = "sofia-taf", full = FALSE)

Arguments

owner

name of a GitHub organization or user.

full

whether to include owner name in output.

Value

String vector of repository names.

Note

This function uses the GitHub API to fetch repository names, using base R.

Author(s)

Arni Magnusson.

See Also

gitClone clones a repository from GitHub.

gitCloneAll clones all SOFIA-TAF repositories from GitHub.

SOFIA-package gives an overview of the package.

Examples

## Not run: 
gitRepos()
gitRepos("arni-magnusson", full=TRUE)

## End(Not run)
